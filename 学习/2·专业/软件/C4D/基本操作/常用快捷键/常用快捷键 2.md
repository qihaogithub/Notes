# 常用快捷键

| 转换成可编辑         | C             |
| -------------- | ------------- |
| 挤压             | D             |
| 向内挤压           | I             |
| 帧选择            | S             |
| 项目设置           | Control+ D    |
| 视图设置           | Shift +V      |
| 调整笔刷大小         | 按住Shift鼠标左右移动 |
| 创建阿诺德材质球并打开编辑器 | ALT+W+N       |
