## 摘录内容 
一，常规的节点缺失排错流程


1，首先用管理器进行安装，点击“安装缺失节点”会自动找到缺失节点，点击安装即可。大概率是可以解决的。

![](https://i0.hdslb.com/bfs/article/0fd32a05468dd837f6ddfc5999d257bc10570268.png@1256w_548h_!web-article-pic.avif)

在管理器安装失败，大概率是梯子问题，请自行解决。

  

  

2，如果管理器找不到。可以拿节点名称去 Github 上搜索，找到项目地址。  
如果你不知道节点名称，可以去 UP 主发布工作流的对应的 Openart 页面，在页面右侧有所有用到的节点和节点组。组名是可以点击直接跳转到 Github 项目地址的。（此办法仅限于发布在 Openart 上的工作流）

![](https://i0.hdslb.com/bfs/article/84e7f03790f4d6c3a336c49bb89fd62510570268.png@!web-article-pic.avif)

有了项目地址后，自己使用 git clone 在命令行内安装，会安装的跳过，不会安装的继续看：  
进入 ComfyUI\_windows\_portable\\ComfyUI\\custom\_nodes 目录，  
在地址栏输入 cmd 回车，启动命令行。  
输入：git clone https://xxxxx然后回车 , (xxxxx 既项目地址)。  
之后会自动拉取代码到本地。  

![](https://i0.hdslb.com/bfs/article/9b4d4724526893017fceb914f0e4bb0910570268.png@1256w_814h_!web-article-pic.avif)

  

  

3，装完之后，进入刚安装的文件夹。现在来安装一下需要的 Python 模块。

如果发现有一个叫 install. Bat 的文件，双击运行它，会自动安装需要的 Python 模块。

![](https://i0.hdslb.com/bfs/article/51819a7f23ae9d06df7f7990c03fd5e410570268.png@!web-article-pic.avif)

  

  

  

  

  

  

4，如果没有 install. Bat，看看有没有一个叫 requirements. Txt 的文件，里面会写明它需要的 Python 模块。（若两个文件都没有，则说明该节点无需新模块，直接重启 Comfyui 运行）

![](https://i0.hdslb.com/bfs/article/4010b786b3d3e0f3dfcdd94ae54e005010570268.png@!web-article-pic.avif)

  

5，接下来要手动安装 requirements. Txt，在当前文件夹的地址栏输入 CMD

![](https://i0.hdslb.com/bfs/article/9696213af080806bd64e320fd3ecb52910570268.png@1256w_342h_!web-article-pic.avif)

  

6，在打开的 cmd 窗口内输入：

`E:\ComfyUI-aki\ComfyUI-aki-v1.3\python\python.exe -m pip install -r requirements.txt`
E:\ComfyUI-aki\ComfyUI-aki-v1.3\python\python.exe 需要替换为实际路径

> 这个命令的意思是使用 Python 的包管理器 pip 来安装 requirements.txt 文件中列出的所有依赖项。
> 1. E:\ComfyUI-aki\ComfyUI-aki-v1.3\python\python.exe: 这是 Python 解释器的完整路径。看起来你正在使用一个名为 ComfyUI-aki 的特定 Python 环境。
> - -m pip: 这告诉 Python 运行 pip 模块。pip 是 Python 的包管理工具。
> - install: 这是 pip 的子命令，用于安装包。
> - -r requirements.txt:
> - -r 选项告诉 pip 从一个文件中读取要安装的包列表。
> - requirements.txt 是一个文本文件，通常包含项目所需的所有 Python 包及其版本。


![](https://i0.hdslb.com/bfs/article/ac28b0d988cec8cc608c7677fe77c23810570268.png@1256w_988h_!web-article-pic.avif)

  

4，回车，确保没有标红。即环境安装成功。

![](https://i0.hdslb.com/bfs/article/bb6c5441b77d64451e6840550fdd688f10570268.png@1256w_906h_!web-article-pic.avif)
