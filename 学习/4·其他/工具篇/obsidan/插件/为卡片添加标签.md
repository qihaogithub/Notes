## 批量添加标签
给一个 markdown 文件中的所有卡片统一标签
在文件内的任意位置，按如下方式设置文件标签的格式：

```
{File Tags Line}
{Tag list}
```
或者
```
{File Tags Line}: {Tag list}
```

举例

```
FILE TAGS
Maths School Physics
```
或者

```
FILE TAGS: Maths School Physics
```

`FILE TAGS` 可以在插件设置中自定义
与标记行格式一样，标记之间需要有一个空格 - 但是，不要包含“Tags:”前缀。
