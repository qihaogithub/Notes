```css
.box {
    width: 100px;
    height: 100px;
    background-color: lightblue;
    transform: scale(0.7); /* 默认缩放比例 */
}

/* 根据不同屏幕宽度动态调整缩放比例 */
@media (max-width: 1200px) {
    .box {
        transform: scale(0.4);
    }
}
@media (max-width: 1400px) {
    .box {
        transform: scale(0.6);
    }
}

@media (max-width: 1900px) {
    .box {
        transform: scale(0.5);
    }
}



```
